# Les Bénéfices d'Alber

Application web (PWA) de suivi des achats, ventes et bénéfices — simple, élégante, installable sur téléphone et PC.

## Fonctions
- Ajout rapide : nom de l'article · prix d'achat · prix de vente (bénéfice calculé automatiquement)
- Optionnel : photo de l'article, nom + téléphone + email + adresse de livraison du client
- Bénéfice total, nombre d'articles, marge moyenne
- Sauvegarde locale sur l'appareil (hors-ligne), export CSV et sauvegarde/restauration
- Thème « Noir & Or »

## Utilisation
Ouvrez `index.html` dans un navigateur, ou visitez le site publié.
Sur mobile : menu du navigateur → **Ajouter à l'écran d'accueil** pour l'installer comme une appli.

## Données
Toutes les données restent sur l'appareil (localStorage). Aucun serveur, aucune collecte.
